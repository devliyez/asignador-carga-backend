## Introduccion

El proyecto de sistema de asignación de carga corresponse al rubro de transporte. Este facilita la asignación de la carga, una vez haya sido aprobado, utilizando un algoritmo que se encarga de buscar un transportista y vehiculo habilitado e ideal para las caracteristicas de la solicitud.

De esta forma se busca agilizar y optimizar los procesos de asignación de una solicitud de carga.


## Ejecución

Para ejecutar este proyecto localmente se debe de seguir las siguientes instrucciones:

- Crear una base de datos en PostgreSQL
- Usar ese nombre de BD para reemplazarlo en application.properties
- Colocar las credenciales(user y password) para conectar con la Base de Datos.

Ejecutar la aplicación.
